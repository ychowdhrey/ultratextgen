# UltraTextGen

UltraTextGen to szybki i przejrzysty generator tekstu Unicode, stworzony dla osób, które nie chcą marnować czasu.

- Żadnych ciężkich stron.
- Żadnych zbędnych ładowań.
- Żadnych rozpraszaczy.

Wpisz tekst, skopiuj go i działaj dalej.


## Dlaczego UltraTextGen

Większość narzędzi do stylizowania tekstu jest powolna, zagracona i przeładowana efektami, których nigdy nie użyjesz.

UltraTextGen jest dokładnie odwrotny.

Ładuje się natychmiast  
Działa w całości w przeglądarce  
Bez rejestracji, bez wyskakujących okien  
Bez obrazków udających tekst

Powstał po to, by oszczędzać czas — nie pochłaniać go.


## Co Robi

UltraTextGen konwertuje zwykły tekst na style oparte na Unicode, które działają na platformach społecznościowych i komunikatorach.

Otrzymujesz tekst, który wygląda na ostylowany, ale po skopiowaniu i wklejeniu zachowuje się jak zwykły tekst.


## Dla Kogo Jest

Twórcy aktualizujący bio i opisy  
Menedżerowie mediów społecznościowych testujący warianty  
Społeczności Discord i graczy  
Każdy, kto chce wizualnego wyróżnienia bez zbędnych komplikacji

Jeśli liczy się szybkość, to narzędzie jest dla Ciebie.


## Zbudowany dla Szybkości

UltraTextGen jest celowo lekki.

- Żadnych frameworków
- Żadnych rozbudowanych skryptów
- Żadnych zbędnych animacji

Wszystko jest zaprojektowane tak, by ładować się szybko i pozostawać responsywnym nawet przy wolnych połączeniach.


## Filozofia Designu

- Szybkość ponad efektowność

- Użyteczność ponad wrażenie

Jeśli coś nie pomaga użytkownikowi działać szybciej, nie trafia do produktu.

## Struktura Projektu

```
ultratextgen/
├── index.html              # Główna strona startowa
├── style.css               # Globalny arkusz stylów
├── script.js               # Główna logika aplikacji
├── styles.js               # Definicje stylów czcionek Unicode
├── renderer.js             # Silnik renderowania tekstu
├── fonts.json              # Dane mapowania czcionek
├── sitemap.xml             # Mapa witryny generowana automatycznie
├── robots.txt              # Dyrektywy dla wyszukiwarek
├── _redirects              # Reguły przekierowań URL
├── scripts/
│   ├── inject-faq-jsonld.js    # Wstrzykiwacz danych strukturalnych FAQ
```

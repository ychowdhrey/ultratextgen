# UltraTextGen

UltraTextGen ist ein schneller und klarer Unicode-Textgenerator, der für Menschen entwickelt wurde, die keine Zeit verschwenden wollen.

- Keine schweren Seiten.
- Keine unnötigen Ladezeiten.
- Keine Ablenkungen.

Schreib deinen Text, kopiere ihn und mach weiter.


## Warum UltraTextGen

Die meisten Fancy-Text-Tools sind langsam, unübersichtlich und vollgepackt mit Effekten, die du nie verwenden wirst.

UltraTextGen ist das Gegenteil.

Lädt sofort  
Läuft vollständig im Browser  
Keine Anmeldung, keine Popups  
Keine Bilder, die als Text getarnt sind

Es wurde entwickelt, um Zeit zu sparen, nicht sie zu verbrauchen.


## Was Es Macht

UltraTextGen wandelt normalen Text in Unicode-basierte Stile um, die auf sozialen Plattformen und Messaging-Apps funktionieren.

Du erhältst Text, der gestylt aussieht, sich aber beim Kopieren und Einfügen wie normaler Text verhält.


## Für Wen Es Gedacht Ist

Kreative, die Bios und Bildunterschriften aktualisieren  
Social-Media-Manager, die Varianten testen  
Discord- und Gaming-Communities  
Alle, die visuellen Nachdruck ohne Aufwand wollen

Wenn Geschwindigkeit zählt, passt dieses Tool.


## Für Geschwindigkeit Gebaut

UltraTextGen ist bewusst leichtgewichtig gehalten.

- Keine Frameworks
- Keine aufgeblähten Skripte
- Keine unnötigen Animationen

Alles ist darauf ausgelegt, schnell zu laden und auch bei langsamen Verbindungen reaktionsfähig zu bleiben.


## Design-Philosophie

- Schnelligkeit über Optik
- Klarheit über Cleverness
- Nützlichkeit über Beeindruckung

Wenn etwas dem Nutzer nicht hilft, schneller voranzukommen, kommt es nicht rein.

## Projektstruktur

```
ultratextgen/
├── index.html              # Haupt-Startseite
├── style.css               # Globales Stylesheet
├── script.js               # Kernlogik der Anwendung
├── styles.js               # Unicode-Schriftstil-Definitionen
├── renderer.js             # Text-Rendering-Engine
├── fonts.json              # Schrift-Mapping-Daten
├── sitemap.xml             # Automatisch generierte Sitemap
├── robots.txt              # Anweisungen für Suchmaschinen
├── _redirects              # URL-Weiterleitungsregeln
├── scripts/
│   ├── inject-faq-jsonld.js    # FAQ-Strukturdaten-Injektor
│   └── update-sitemap.js       # Sitemap-Generierungsskript
├── category/               # Schriftkategorie-Seiten
├── usecase/                # Anwendungsfall-Seiten
├── guide/                  # Redaktionelle Guide-Seiten
├── discord/                # Discord-Plattformseite
├── facebook/               # Facebook-Plattformseite
├── instagram/              # Instagram-Plattformseite
```

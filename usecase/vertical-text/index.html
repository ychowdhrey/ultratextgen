Vertical Text FAQ

Vertical text places each letter on its own line, creating a top-to-bottom reading direction. It's popular in social media bios, aesthetic captions, and creative text art.
Choose a layout from the options: Simple stacks letters vertically, Extra Spaced adds breathing room, Box wraps each letter in a box frame, Staircase creates indented steps, and Double Column puts two words side by side.
In Stack mode, each word is separated by a blank line. In Continuous mode, spaces are removed and all letters flow as a single vertical stream.
Prefix decorators (bullets, arrows, dots, emojis, flags) add a symbol before each letter line. Divider decorators insert a separator line between letter lines. Pick "None" to remove all decorations.

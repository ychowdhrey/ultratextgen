Lists and Quotes in WhatsApp

Add an asterisk or hyphen followed by a space before each line. 
* Item one
* Item two
Start each line with a number, a period, and a space. 
1. First item
2. Second item
Place an angle bracket and a space before the text. 
> This is a quoted message

What Font Does WhatsApp Use?

WhatsApp uses your device's default system font. You cannot change it, but here is what renders on each platform:

  • iPhone / iPad: San Francisco (SF Pro)
  • Android: Roboto
  • Windows Desktop: Segoe UI
  • Mac Desktop: Helvetica Neue
  • WhatsApp Web: Your browser's default system font

While you cannot swap the base font, you can use WhatsApp's built-in formatting shortcuts for bold, italic, strikethrough, and monospace — or generate Unicode-styled text with the tool above.

WhatsApp Font Shortcuts and Settings

WhatsApp supports these formatting shortcuts:

• *text* for bold
• _text_ for italic
• ~text~ for strikethrough
• ```text``` for monospace
• `text` for inline code
• > for quotes
• * or - for bulleted lists
• 1. for numbered lists

Combine them for mixed styles — for example, _*text*_ creates bold italic text.

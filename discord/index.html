Discord Nicknames

A server nickname is a name set within a specific Discord server. It overrides your display name in that server only. For example, if your display name is "Phibi" but your server nickname is "Wumpus", you'll appear as "Wumpus" in that server and "Phibi" everywhere else. You can set a server nickname by going to User Settings > Profiles > Per-server Profiles.
A friend nickname is a name you assign to another user. It overrides their display name in your view. You can set a friend nickname by right-clicking on their profile avatar and selecting Add Friend Nickname.
The priority order from highest to lowest is: Friend nickname (if set by you for that user), then Server nickname (if set in that specific server), then Display name (shown everywhere else). The Username is used as a unique identifier and shown alongside the display name for verification purposes.

Discord Chat Formatting

Discord uses Markdown for chat formatting. Wrap text in **double asterisks** for bold, *single asterisks* or _underscores_ for italic, __double underscores__ for underline, ~~double tildes~~ for strikethrough, ***triple asterisks*** for bold italic, and ||double pipes|| for spoiler text.
Use the hash symbol at the start of a new line followed by a space. Use # for a big header, ## for a medium header, or ### for a small header. You can also use -# at the beginning of a line to add subtext.
Wrap text in single backticks (`) for inline code. For multi-line code blocks, wrap your text in triple backticks (```) on their own lines before and after the code.
Wrap your text in double pipe characters: ||spoiler text||. The text will be hidden until the reader clicks on it. You can also highlight text and click the eye icon, or type /spoiler before your message. Note that text marked as spoiler within code blocks will not be hidden.
Yes. For images, click the Mark as Spoiler eye icon on the attachment before sending. For links, wrap the URL in double pipes: ||https://example.com||. Note that Discord server invite embeds cannot be hidden with spoiler tags.
Start each line with a dash (-) or asterisk (*) followed by a space. To indent items, add 2 spaces before the dash or asterisk at the beginning of each line.
Use > followed by a space at the start of a line for a single-line block quote. Use >>> followed by a space before the first line to create a multi-line block quote that covers everything after it.
Use the format [displayed text](URL). For example, [Click here](https://example.com) will show "Click here" as a clickable hyperlink.

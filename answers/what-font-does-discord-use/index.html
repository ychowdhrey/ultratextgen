What font does Discord use?

The current UI font, the previous one, and the practical distinction most users miss.

Verdict

Discord uses gg sans, its proprietary typeface, and has done so since 2023. Before gg sans, Discord's interface relied on a customized version of Whitney.

So if you are asking about Discord's buttons, menus, and app UI, the answer is gg sans.

Availability

Can you download or install Discord's font?

No. gg sans is Discord's proprietary product typeface and is not released as a public download for general design use.

Designers looking for similar UI tone typically use clean geometric or humanist sans-serif alternatives, but there is no official public gg sans package.

Important distinction

Discord UI font vs styled text in messages

Discord's UI font choice (gg sans) is separate from the “fancy fonts” users paste into messages and bios. Those user-visible styles are Unicode characters, not a Discord font setting.

That is why people can style messages on free accounts regardless of UI typography. For the Nitro question and compatibility caveats, see Do you need Nitro for Discord fonts?.

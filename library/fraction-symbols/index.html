Fraction Symbols

Every precomposed Unicode fraction — halves, thirds, quarters, eighths, and beyond — plus the slash and digits to build any fraction you need. Click any symbol to copy it instantly.

Vulgar (precomposed) fraction characters render a complete fraction in a single glyph. This library collects the common halves, thirds, and quarters, the eighths, fifths, sixths, sevenths, ninths, and tenths, and a build-your-own set using the fraction slash with superscript and subscript digits.

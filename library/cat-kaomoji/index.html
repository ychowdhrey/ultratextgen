Cat kaomoji (also spelled cat kamoji) are Japanese-style text faces built around the classic whisker shape =^・ω・^= — the most expressive single-animal family of kaomoji. A cat kaomoji like (=^・ω・^=) or ฅ^•ﻌ•^ฅ is plain Unicode text, so it pastes cleanly into bios, captions, comments, and usernames.

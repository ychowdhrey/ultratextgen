Vertical Line Symbols

Every pipe, bar, and vertical rule Unicode offers — from the keyboard | to heavy ❚ bars. Click any symbol to copy it instantly.

The vertical line symbol — also called the pipe, bar, or vertical slash — separates items in headlines, bios, navigation menus, and code. Beyond the keyboard pipe |, Unicode includes double bars, dashed box-drawing verticals, heavy dingbat bars, and mathematical divides-by characters. Pick the exact weight and style of vertical line you need below; each one is a single copyable character.

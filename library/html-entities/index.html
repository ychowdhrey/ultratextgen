HTML Entities

Named HTML entity codes and the characters they produce — reserved characters, punctuation, math, and more. Click any character to copy it instantly.

HTML entities are named codes like &amp; and &copy; that let you write special characters safely in web markup — essential for the reserved characters &, <, and >, and convenient for symbols that are hard to type. This reference lists the most useful HTML entities grouped by category, with each character labeled by its entity name. Click a character to copy the symbol itself, or type the entity code shown in its label into your HTML.

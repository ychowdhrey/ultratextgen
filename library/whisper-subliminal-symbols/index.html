About Diacritics

About Combining Diacritical Marks

Many symbols in this collection are combining diacritical marks (Unicode category 'Mn' — non-spacing marks). They don't stand alone — they attach to the preceding character. This is intentional: in whisper aesthetics, they create a floating-dot effect around text. If you copy a combining mark and paste it at the start of a line, it may attach to whatever character precedes it in that context. Test in a notes app first to see how it renders.

Minecraft Symbols

Chat brackets, MOTD shading blocks, box-drawing lines, and game-themed icons that render in Minecraft's font. Click any symbol to copy it instantly.

Minecraft's default font renders a wide slice of Unicode, which is why servers decorate MOTDs, chat ranks, holograms, and kick screens with characters like ►, ▌, and ☠. This library collects the Minecraft symbols that show up reliably in chat, server list entries, scoreboard lines, and plugin configs — from shading blocks for MOTD banners to pickaxe and sword icons for rank prefixes. Paste them into server.properties-driven MOTD plugins, essentials chat formats, or anvil-renamed items.

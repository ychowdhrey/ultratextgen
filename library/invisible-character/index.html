Invisible Character

Every invisible character and blank space Unicode offers — zero width spaces, fillers, and empty braille. Click any symbol to copy it instantly.

An invisible character is a real Unicode character that renders as nothing — a blank space you can copy and paste where platforms normally reject empty input. People use them to make an invisible Discord name, add blank lines to an Instagram bio or caption, send a seemingly empty WhatsApp message, or create clean spacing in gaming nicknames. Every button below holds a genuine invisible or blank character: the tiles look empty, but the labels tell you exactly which character you are copying.

Keyboard Symbols

Command, option, shift, and every other Mac and PC key glyph for writing clean shortcuts. Click any symbol to copy it instantly.

Documenting keyboard shortcuts reads far better with the real key glyphs than with spelled-out names. This library gathers the Mac modifier symbols such as command (⌘), option (⌥), shift (⇧), and control (⌃); the editing and navigation keys like return, backspace, and tab; and the arrow and special-key glyphs that complete a shortcut. Click any character to copy it for documentation, tutorials, or app interfaces.

Caractères spéciaux pseudo : les règles par jeu

— Fortnite : 16 caractères max pour le nom d'affichage Epic ; ꧁꧂, 亗, シ et les lettres stylées passent ; changement possible toutes les 2 semaines.
— Free Fire : 12 caractères ; changement de pseudo via carte de renommage ou diamants — vérifie la longueur avant de payer.
— Roblox : le nom d'affichage accepte les caractères Unicode, le nom d'utilisateur (@) non.
— Discord : quasiment tout passe, y compris le zalgo — le pseudo de serveur est modifiable à volonté.
— TikTok / Instagram : le nom affiché accepte les styles, le @ n'accepte que lettres simples, chiffres, points et tirets bas.

Copie ton pseudo, colle-le dans le champ du jeu, garde-le sous la limite de caractères — et souviens-toi que certaines lettres stylées comptent double.

Prénoms accentués et pseudo stylé

Les jeux de lettres stylées d'Unicode couvrent l'alphabet latin de base (a–z) : é, è, ê, à et ç n'ont pas d'équivalent stylé. Écris «Léa» et le é restera droit au milieu du pseudo stylé. L'astuce des joueurs : simplifier le prénom — Léa → Lea → ꧁𝓛𝓮𝓪꧂, Théo → Theo → メ𝐓𝐇𝐄𝐎彡.

Autre approche qui rend très bien : garder le nom en lettres simples et ne styler que le cadre — ꧁ Léa ꧂ — lisible partout, accents compris. Pour une initiale décorée, pioche une majuscule dans l'alphabet calligraphie.

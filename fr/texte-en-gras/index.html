Le gras Unicode : un gras qui se colle partout

Le texte en gras de cette page n'est pas une mise en forme comme dans Word — chaque lettre est un caractère Unicode dont le dessin est déjà gras. Résultat : quand tu copies, le gras voyage avec le texte. Il tient dans une bio Instagram, un statut WhatsApp, une description TikTok ou un pseudo de jeu — tous ces endroits où il n'existe aucun bouton «G».

Exemple : «bonjour» → 𝗯𝗼𝗻𝗷𝗼𝘂𝗿 (sans bold), 𝐛𝐨𝐧𝐣𝐨𝐮𝐫 (serif bold), 𝙗𝙤𝙣𝙟𝙤𝙪𝙧 (gras italique)

Le gras à astérisques de WhatsApp (*comme ça*) ne fonctionne que dans le message ; dès que tu le copies ailleurs, il redevient plat. Le gras Unicode, lui, est le caractère — où que tu le colles, il reste gras. Pour comparer avec les autres familles, va voir la page police d'écriture.

Lequel choisir ? Le sans bold est net et moderne — le standard des bios et des titres. Le serif bold a un côté manchette de journal, plus classique. Le gras italique est à la fois épais et penché — la version la plus assumée. Dans le doute, tape ton mot en haut et compare les trois côte à côte.

Écrire en gras : Facebook, Instagram, WhatsApp, LinkedIn

Chaque plateforme a ses (non-)options de mise en forme. Voilà comment le gras Unicode s'en sort partout :

— Facebook : aucun bouton gras dans les posts et commentaires — le gras Unicode est la seule option qui marche partout
— Instagram : bio, légende, commentaire — tout accepte le gras Unicode ; seul le @ reste en lettres simples
— WhatsApp : *astérisques* dans les messages, gras Unicode pour le statut, le nom et les infos de groupe
— LinkedIn : un titre en 𝗴𝗿𝗮𝘀 attire l'œil dans le fil — sans casser la recherche si tu gardes ton nom en lettres normales
— Objet de mail : les objets sont en texte brut — le gras Unicode s'y affiche alors que celui d'Outlook ou Gmail saute

Un conseil qui vaut pour tout : mets un mot en gras, pas la phrase entière. C'est le contraste qui attire l'œil — si tout est gras, plus rien ne ressort. Pour l'effet inverse — discret et minimal — regarde l'écriture italique.

Questions sur le texte en gras

Tape ton texte dans la barre en haut de la page — les versions sans bold (𝗴𝗿𝗮𝘀), serif bold (𝐠𝐫𝐚𝐬) et gras italique (𝙜𝙧𝙖𝙨) s'affichent instantanément. Clique sur «Copier» et colle où tu veux. Comme ce sont de vrais caractères Unicode, le gras reste intact même dans les champs qui n'ont aucun bouton de mise en forme.
Facebook n'a pas de bouton gras pour les posts et les commentaires. La solution : tape ton texte ici, copie la version en gras Unicode et colle-la dans ton post, ton commentaire ou ta bio Facebook. Le gras s'affiche partout — fil d'actualité, groupes, pages — sans extension ni application.
Instagram ne propose aucune mise en forme : ni dans la bio, ni dans les légendes, ni dans les commentaires. Le gras Unicode passe partout : copie ton texte en 𝗴𝗿𝗮𝘀 depuis cette page et colle-le. Seul le nom d'utilisateur (@) refuse les caractères spéciaux — il n'accepte que lettres simples, chiffres, points et tirets bas.
Entourer un mot d'astérisques (*gras*) fonctionne dans un message WhatsApp, mais cette mise en forme disparaît dès que tu la copies ailleurs — et elle ne marche ni dans ton statut ni dans ton nom. Le gras Unicode est différent : chaque lettre est déjà dessinée en gras, donc il se colle partout — statut, nom de groupe, bio Insta, pseudo de jeu.
Partiellement. Le jeu de caractères gras d'Unicode couvre l'alphabet latin de base ; é, è, à, ç, ù et î restent en graisse normale au milieu du mot. Pour un rendu 100 % gras, écris sans accents : «été» → ete → 𝗲𝘁𝗲. Sinon, garde les accents — le mot reste parfaitement lisible avec quelques lettres non grasses.

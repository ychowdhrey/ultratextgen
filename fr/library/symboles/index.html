Comment copier

Comment copier-coller un symbole ?

Touche le symbole → copié. Puis appui long dans le champ de texte et «Coller» (Ctrl+V sur ordinateur). Ça marche dans les pseudos, les bios, les statuts et tout champ qui accepte du texte.

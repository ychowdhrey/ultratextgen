What Font Does TikTok Use?

Short answer first, then the full breakdown: TikTok Sans, legacy naming confusion, in-app editor fonts, and free lookalikes for creators.

Quick answer

TikTok's current interface font is TikTok Sans, rolled out in 2023. It replaced the earlier default look most users associated with "Classic" or Proxima Nova-era UI styling.

For off-platform design work, you can use TikTok Sans directly (open source), or near-matches like Montserrat and Poppins when you need quick alternatives in common design tools.

About TikTok Sans (and download status)

TikTok Sans is TikTok's custom brand typeface intended for global readability across app surfaces. TikTok released it as an open-source family under the SIL OFL license, which means creators and designers can use it outside the app.

If your workflow uses templates and brand kits, installing TikTok Sans can make social graphics feel closer to native TikTok UI tone than default system sans fonts.

Why people confuse TikTok fonts

Misconception

"TikTok font" means one thing

  • It can mean UI typeface (TikTok Sans), editor presets, or Unicode copy-paste text.
Misconception

"Classic" equals current default

  • "Classic" often refers to in-app style labels or older visual memory, not today's full UI typography stack.
Misconception

Styled bios are a TikTok font setting

  • They're Unicode characters pasted into text fields, not a built-in font picker for profile text.
